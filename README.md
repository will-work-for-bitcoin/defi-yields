# defi-yields

DeFi yield aggregator CLI - compare APY across Aave, Compound, Lido, and more protocols.

## Installation

```bash
git clone https://github.com/will-work-for-bitcoin/defi-yields.git
cd defi-yields
```

## Usage

```bash
python defi_yields.py
python defi_yields.py --help
```

## Features

- Pure Python 3.7+, zero dependencies
- JSON/CSV export support
- Beautiful terminal output
- Easy to extend and customize
